Создать телеграм бота, который будет помогать искать музыку по строке/названию, сайт и интерактивную игру.
Бот должен искать музыку и предлагать на выбор функции:
1. Показать текст песни
2. Показать автора песни
3. Показать название песни
4. Показать ссылку на песню в спотифае
Присутствует база данных, с несколькими таблицами, для:
1. Пользователи
2. Песни
3. Авторы
4. Запросы

====================================

Сайт имеет вкладки:
1. Главная страница
2. Песни
3. Авторы
4. Таблица лидеров
Для авторизированных пользователей сайт должен показывать историю поиска.
